# Джун коммитит в гит

## Называет ветки, как попало

❌ **Плохо:**

```
git checkout -b "my-branch-123"
```

✅ **Хорошо:**

```
git checkout -b "feat/TASK-523-update-user-list"
```

## Commit message отсутствуют, либо называются типа «fix»

❌ **Плохо:**

```
git commit -m "fix"
```

✅ **Хорошо:**

```
git commit -m "fix add users button bug"
```
